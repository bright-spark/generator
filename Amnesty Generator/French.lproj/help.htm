Google Gadgets

Une vaste bibliothèque de widgets (Google les appelent des gadgets) conçue pour utilisation sur n'importe quelle page Web.

Le page d'accueil par défaut affiche la plupart des gadgets populaires de Google, mais vous pouvez choisir d'autres groupes à partir du menu à gauche. Vous pouvez également rechercher un widget en utilisant la barre de recherche au dessus de la page. Cliquez sur le nom ou l'image des widgets pour obtenir plus d'informations sur celui-ci.

Instructions :
1) Cliquez sur Ajouter votre page Web.
2) Définissez tous les réglages nécessaires.
3) Cliquez sur Obtenir le code.
4) Copiez le code HTML de la case qui apparaît au bas de la page
5) Collez le code dans Amnesty Generator.
